# Arbeitsbericht PWA

- Offline-fähige App-Shell (Service Worker)
- IndexedDB für lokale Reports
- Sync-Endpoint: `api/report_sync.php`
- Serverseitige PDF-Erzeugung: **ja, nach erfolgreicher Synchronisierung** (geplant)

## Dev-Hinweise
- Webroot: `/public`
- HTTPS erforderlich
